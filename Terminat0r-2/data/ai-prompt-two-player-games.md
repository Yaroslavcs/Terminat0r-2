# AI промт: генерація ігор удвох на телефоні/вдома

Використовуй цей промт для AI (Gemini, ChatGPT, Claude), щоб отримати додаткові ідеї ігор для двох людей.

---

## Промт

```
Ти експерт з ігор та дозвілля. Згенеруй 20–50 ігор, які можна грати удвох:
- на телефоні (один або два телефони)
- вдома без спеціального інвентарю
- 5–30 хвилин на гру
- для дорослих або сім'ї

Формат для кожної гри:
1. Назва
2. Що потрібно (телефон, папір, олівець, нічого)
3. Правила (коротко, 2–4 речення)
4. Переможець / мета
5. Варіація (опціонально)

Мова: українська. Стиль: дружній, зрозумілий. Ігри мають бути креативними, не лише "камінь-ножиці-папір".
```

---

## Приклад відповіді (фрагмент)

| Назва | Інвентар | Правила | Мета |
|-------|-----------|---------|------|
| Швидкий поет | 2 телефони | Кожен відкриває словник/генератор слів. За 2 хв скласти вірш з 3 слів. Читаєте вголос. | Кращий вірш за голосування |
| Міміка-марафон | Нічого | Один показує без слів (фільм, пісня, емоція). Другий вгадує. Змінюєтесь. | Хто швидше вгадає 5 — виграв |
| Список за 60 | Папір, олівець | Тема: "тварини", "країни", "їжа". За 60 сек кожен пише список. Унікальні = 1 бал. | Більше балів |

---

## Розширений промт (для більше контексту)

```
Контекст: додаток Terminat0r-2 — мобільний додаток для продуктивності, здоров'я, рутини. Користувачі можуть бути поруч (Bluetooth), і додаток показує поради для спільної гри.

Згенеруй ігри, які:
- поєднують продуктивність і веселощі (наприклад: "хто швидше прибере стіл")
- можна грати без підготовки
- підходять для пари, друзів, сім'ї
- тривалість 5–30 хв
- не потребують інтернету (опціонально)

Виведи у форматі JSON:
{
  "games": [
    {
      "name": "Назва",
      "duration_min": 10,
      "materials": ["телефон", "папір"],
      "rules": "Короткі правила",
      "goal": "Мета гри",
      "category": "productivity|fun|creative|sport"
    }
  ]
}
```
