# маскоти для рівнів

згенеровано 20 зображень маскота (біло-синій робот-єнот) у різних костюмах для рівнів 1–20.

## костюми

| рівень | тема | костюм |
|--------|------|--------|
| 1 | starter | базовий робо-костюм |
| 2 | explorer | капелюх дослідника, рюкзак, компас |
| 3 | warrior | броня, меч, шолом |
| 4 | mage | мантія магії, капелюх, посох |
| 5 | chef | кухарський капелюх, фартух, лопатка |
| 6 | scholar | окуляри, шапка випускника, книги |
| 7 | athlete | спортивна форма, кросівки, медаль |
| 8 | gardener | соломʼяний капелюх, комбінезон, лейка |
| 9 | artist | берет, палітра, пензель |
| 10 | merchant | жилет, гаманець, ваги |
| 11 | healer | біла мантія, хрест, трави |
| 12 | ranger | капюшон, лук, колчан |
| 13 | blacksmith | шкіряний фартух, молот, ковадло |
| 14 | bard | капелюх з пірʼям, лютня, сувій |
| 15 | alchemist | лабораторний халат, зілля, колба |
| 16 | knight | повна броня, щит, меч |
| 17 | rogue | темний плащ, кинджали, маска |
| 18 | druid | мантія з листя, посох, лози |
| 19 | monk | проста мантія, чотки |
| 20 | paladin | свята броня, світячий щит |

## файли

зображення збережені в `app/assets/mascots/`:

- mascot-01-starter.png … mascot-20-paladin.png

екран першого запуску (`first-launch.tsx`) показує зображення маскота замість emoji.

## івент first_launch

при першому запуску показується екран з маскотом поточного рівня. дані: `data/first-launch-event.json`, `data/level-mascots.json`
